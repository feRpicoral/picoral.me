## O que faz

Times fazem upload dos seus documentos — PDFs, Word, Markdown, transcrições — e conversam com eles em linguagem natural. Cada resposta é embasada por citações inline. Clicar em uma citação abre o documento-fonte exatamente na região que gerou a resposta, com o trecho relevante destacado.

## Por que construí

Não é um produto — é um projeto pessoal. Chatbots RAG sem citação são infalsificáveis. Embasar as respostas em citações torna o output do modelo verificável, que é justamente o ponto de usar um sistema de recuperação. Também queria criar uma UI onde a citação é a interação principal, não uma nota de rodapé — o visualizador de documentos tem o mesmo peso que o chat.

## Status

Em desenvolvimento ativo. O pipeline de recuperação e a UI de chat funcionam de ponta a ponta; o foco atual é a lógica de âncoras de região do visualizador de documentos.
