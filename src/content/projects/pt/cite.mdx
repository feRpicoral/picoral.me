## O que faz

Times fazem upload de documentos — PDFs, Word, Markdown, transcrições — e conversam com eles em linguagem natural. Cada resposta é fundamentada por citações inline. Clicar em uma citação abre o documento-fonte na região exata que gerou a resposta, com o trecho relevante destacado.

## Por que construí

Não é um produto — é um projeto pessoal. Chatbots RAG sem citação são infalsificáveis. A fundamentação por citações torna o output do modelo verificável, que é o ponto central de usar um sistema de recuperação. Também queria criar uma UI onde a citação é a interação principal, não uma nota de rodapé — o visualizador de documentos é tão central quanto o chat.

## Status

Em desenvolvimento ativo. O pipeline de recuperação e a UI de chat estão funcionando de ponta a ponta; a lógica de âncora de região do visualizador de documentos é o foco atual.
