Uma pequena utilidade open-source que criei para resolver um problema pessoal. O r/Visible tem um megathread quinzenal onde os usuários postam códigos de indicação; o meu ficava desatualizado antes de expirar. O bot resolve o atrito com um GitHub Action agendado via cron.

Duas variantes — uma usando PRAW, outra usando um navegador Chromium headless via Playwright para casos em que o fluxo de autenticação do Reddit bloqueia tokens de API. Ambas escrevem um arquivo de estado para rastrear o último thread postado, garantindo idempotência após reinícios.
