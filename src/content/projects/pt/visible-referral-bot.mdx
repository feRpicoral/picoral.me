Um utilitário open-source que criei para resolver um problema meu. O r/Visible tem um megathread quinzenal onde os usuários postam códigos de indicação; o meu sempre expirava antes de ser usado. O bot resolve isso com um GitHub Action agendado via cron.

Duas variantes — uma usando PRAW, outra com um navegador Chromium headless via Playwright, para os casos em que o fluxo de autenticação do Reddit bloqueia tokens de API. As duas gravam um arquivo de estado para rastrear o último thread postado, garantindo idempotência entre reinicializações.
