Um utilitário open-source pequeno que criei pra resolver um problema meu. O r/Visible tem um megathread quinzenal onde os usuários postam códigos de indicação; o meu ficava expirando antes do prazo. O bot resolve isso com um GitHub Action agendado via cron.

Duas variantes — uma usando PRAW, outra usando um navegador Chromium headless via Playwright para os casos em que o fluxo de autenticação do Reddit bloqueia tokens de API. As duas gravam um arquivo de estado para rastrear o último thread postado, garantindo idempotência entre reinicializações.
