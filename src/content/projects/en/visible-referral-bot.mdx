A small open-source utility I built to scratch my own itch. r/Visible runs a biweekly megathread where users drop referral codes; mine kept aging out before it would expire. The bot solves the friction with a cron-scheduled GitHub Action.

Two variants — one using PRAW, one using a headless Chromium browser via Playwright for cases where Reddit's auth flow blocks API tokens. Both write a state file to track the last-posted thread so they're idempotent across restarts.
