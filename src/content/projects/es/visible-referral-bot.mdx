Una pequeña utilidad open-source que hice para resolver mi propio problema. r/Visible tiene un megahilo quincenal donde los usuarios publican códigos de referido; el mío siempre vencía antes de expirar. El bot elimina esa fricción con un GitHub Action programado por cron.

Dos variantes — una usando PRAW y otra usando un navegador Chromium headless vía Playwright, para los casos en que el flujo de autenticación de Reddit bloquea los tokens de API. Ambas escriben un archivo de estado para registrar el último hilo publicado y así ser idempotentes entre reinicios.
