Una pequeña utilidad de código abierto que construí para resolver un problema personal. r/Visible tiene un megathread quincenal donde los usuarios publican códigos de referido; los míos caducaban antes de que se vencieran. El bot resuelve la fricción con una GitHub Action programada como cron.

Dos variantes —una usa PRAW y la otra un navegador Chromium headless vía Playwright para casos donde el flujo de autenticación de Reddit bloquea los tokens de API. Ambas escriben un archivo de estado para rastrear el último thread publicado, de modo que sean idempotentes tras reinicios.
